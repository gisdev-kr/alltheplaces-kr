WHAT IS ALL THE PLACES KR?

한국 POI를 수집하고,
열린 스키마로 정리합니다.

All the Places KR는 한국에서 영업하는 프랜차이즈와 시설의 공개 위치 정보를 수집하는 프로젝트입니다. upstream All the Places의 Scrapy 구조, item model, 정제 pipeline, NSI matching과 exporter를 가능한 그대로 사용합니다.

Name Suggestion Index의 한국 영업 범위와 OSM Schema를 기준으로 name, brand:wikidata, shop, amenity, 주소, 전화, 웹사이트, 좌표를 일관된 형식으로 정리합니다.

생성된 OSM 파일은 자동 업로드하지 않으며, mapping과 conflation 결과는 사람이 검토하기 위한 자료입니다.
